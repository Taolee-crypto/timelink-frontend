TL-AGENT 시스템 설명

TL-AGENT 아키텍처

기존 MP3 플레이어와 호환되는 시간 기반 수익화 시스템

시스템 작동 원리

  1. MP3 메타데이터 확장: 기존 MP3 파일의 ID3 태그에 TL 소비 정보 추가
  2. TL-AGENT 설치: 사용자 기기에 경량 에이전트 설치 (브라우저 확장/모바일 앱)
  3. 자동 감지: TL-AGENT가 TL-MP3 재생을 감지하고 TL 소비 시작
  4. 실시간 분배: 소비된 TL은 블록체인 기반으로 실시간 분배

TL(Time Link) 토큰

TL은 음악 청취 시간을 측정하는 단위입니다. 1 TL = 1초의 청취 시간에 해당합니다.

  • 사용자는 TL을 구매하거나 활동으로 획득
  • 음악 재생 시 실시간으로 TL 소비
  • 소비된 TL은 수익으로 전환되어 분배
  • TL 잔액이 없으면 재생 중지
