TimeGate 변환 시스템 개요

파일을 카테고리별로 변환하여 TL(시간 라이선스)을 부착합니다. 각 카테고리별 특징:

  • 음원(TL3): 재생 시간에 따라 TL 자동 계산 (예: 3분 음악 = 180초 TL 소비)
  • 영상(TL4): 재생 시간에 따라 TL 자동 계산 (예: 5분 영상 = 300초 TL 소비)
  • 문서/기타(TLD/TLS): 업로더가 사용 가능 시간 직접 설정, 파일 오픈 시 자동 차감
  • SHAREPLACE 연동: 변환된 모든 파일을 SHAREPLACE에 업로드하여 배포 가능

변환 카테고리 선택

변환할 파일 유형에 맞는 카테고리를 선택하세요

TL3

음원 TL3

모든 음원 대상

음악, 오디오북, 팟캐스트 등 모든 음원 파일에 시간 라이선스(TL3)를 부착합니다. 재생 시간에 따라 TL이 자동으로 계산되어 소비됩니다.

MP3 WAV FLAC AAC M4A OGG
TL = 재생 시간
TL4

영상 TL4

모든 영상 대상

영상 콘텐츠에 시간 라이선스(TL4)를 부착합니다. 재생 시간에 비례하여 TL이 소비되며, 일시정지 시 TL 소비가 중지됩니다.

MP4 AVI MKV MOV WMV WEBM
TL = 재생 시간
TLD

문서 TLD

모든 문서 대상

문서 파일에 시간 라이선스(TLD)를 부착합니다. 업로더가 사용 가능 시간을 설정하며, 문서가 열려있는 동안 설정된 시간 단위로 TL이 소비됩니다.

PDF DOC/DOCX PPT/PPTX XLS/XLSX TXT RTF
업로더 설정

JPG/JPEG PNG GIF BMP SVG WEBP
