새 광고 등록
광고 예산(TL)은 등록 즉시 차감되며, 사용자가 광고를 시청할 때마다 해당 TL이 지급됩니다.
기본 정보
미디어
직접 업로드는 준비 중. R2 URL 직접 입력 가능
예산 설정
최소 1,000 TL · 잔액: -- TL
50~1,000 TL (기본 300 TL)
하루에 노출할 최대 횟수
예산을 입력하면 예상 노출 수를 계산합니다.
🎉
광고가 등록되었습니다!
검토 후 즉시 노출됩니다. 내 광고 탭에서 성과를 확인하세요.
📋 광고 등록 순서
1
광고 유형 선택
리워드(영상·오디오·이미지) 또는 사이드바 배너 중 선택합니다.
2
소재 업로드
영상 MP4(15초), 오디오 MP3(30초), 이미지 JPG/PNG, 배너 220×120px를 준비하세요.
3
예산(TL) 설정
광고 예산을 TL로 설정합니다. 등록 즉시 차감되며 예산 소진 시 자동 종료됩니다.
4
랜딩 URL 입력
광고 클릭 시 이동할 URL (선택사항)
5
등록 즉시 게시
심사 없이 바로 게시됩니다. 내 광고 탭에서 성과를 확인하세요.
⚡ 광고 유형별 단가
📹 영상 15초
리워드 최소 10,000 TL
🎵 오디오 30초
리워드 최소 10,000 TL
🖼️ 이미지 10초
리워드 최소 5,000 TL
📌 사이드바 배너
배너 50,000 TL / 7일
💡 TL 1 = 1원 · 시청자는 광고TL(TL_A) 적립
📐 소재 규격
📹
영상 — MP4 · 최대 50MB · 15초
권장 1080×1920 (세로) / 1920×1080
🎵
오디오 — MP3 · 최대 5MB · 30초
권장 128kbps 이상
🖼️
이미지 — JPG·PNG · 최대 2MB
권장 1080×1080
📌
사이드바 배너 — JPG·PNG · 최대 500KB
필수 220×120px
⚠️ 주의사항
• 허위·과장 광고, 불법 콘텐츠 등록 금지
• 등록 후 예산은 즉시 차감, 환불 불가
• 예산 소진 시 자동 종료 (재충전 가능)
