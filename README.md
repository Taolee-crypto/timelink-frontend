# TIMELINK - AI 음악 TL3 플랫폼

## 개요
TIMELINK는 AI 생성 음악을 TL3(Track-Link) 형식으로 변환하여 시간 기반 경제 시스템에서 유통, 재생, 보상을 제공하는 플랫폼입니다.

## 주요 기능

### 🎵 크리에이터
- AI 음악을 TL3로 변환
- Spotify 인증을 통한 검증
- 재생 시간 충전
- 플랫폼 기여 보상 (환전 가능 TL)

### ☕ 카페/비즈니스
- 저작권 분쟁 없는 음악 방송
- 시간당 TL 소모 기반 비용 구조
- 무드 기반 자동 재생
- TL 사용 전용 (증식 불가)

### 🎧 리스너
- 음악 재생 및 참여
- 재생 시간당 TL 획득
- 광고 시청 보상
- 좋아요/반응 보상

## TL 이코노미

### 💰 TL 획득 (환전 가능)
- 플랫폼 기여 보상 (크리에이터, 리스너 활동)
- 음악 재생 (30초당 1 TL)
- 참여 활동 (좋아요, 반응 등)

### 🎁 TL 획득 (환전 불가)
- 가입 보너스 (10,000 TL)
- 광고 시청 (50 TL)
- TL 구매

### ⚡ TL 사용
- TL3 시간 충전
- 카페 방송 (시간당 ~100 TL)
- 음악 재생
