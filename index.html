1. TimeGate™란 무엇인가

TimeGate™는 파일에 시간(Time)과 권한(Authority)을 결합해
허용된 시간·환경·사용자만 파일을 열 수 있도록 만드는 보안 솔루션입니다.

기존 DRM이 "복사 금지"에 머물렀다면, TimeGate™는 "언제 / 어디서 / 누구에게 / 얼마나"까지를 통제합니다.

4. Converter 사용 설명 (변환기)

1

Converter 실행

TimeGate Converter 실행 후 로그인 (회사 관리자 계정 또는 개인 제작자 계정)

2

파일 불러오기

드래그 앤 드롭 또는 "파일 선택" 버튼 클릭

PDF / DOCX / XLSX
DWG / DXF / STEP
MP3 / WAV / MP4
기타 모든 바이너리 파일

3

TimeGate 설정 (핵심)

• 사용 가능 시간 설정 (시작일/종료일 또는 총 시간)

• 환경 제한 (회사 내부 PC, 특정 디바이스, OS)

• 사용자 권한 (회사 계정, 이메일, SSO 연동)

4

회사별 암호 키 적용

회사 고유 Master Key + 파일별 Session Key

이 키는 외부로 노출되지 않으며
Viewer 없이는 어떤 디버깅도 불가능

5

변환 실행

[변환 시작] 클릭 → 결과 파일 생성

example.pdf → example.tld

✔ 원본 파일은 암호화 + 시간게이트 삽입
✔ 확장자만 바뀌며 내부 구조는 완전히 보호됨

5. Viewer / Player 사용 설명

1

파일 열기

.tld 파일 더블클릭 → 자동으로 TimeGate Viewer 실행

사용자는 아무 설정도 할 필요 없음

2

Viewer 내부 처리 (자동)

• Step 1: 무결성 검사 (파일 위·변조 확인)

• Step 2: 시간 검증 (현재 시간 vs 허용 시간)

• Step 3: 권한 확인 (사용자 인증)

3

검증 성공 시

임시 메모리 공간에서 원본 복호화 후 기존 프로그램 호출

PDF → Adobe Reader
DWG → AutoCAD
MP4 → OS 기본 플레이어

⚠️ 파일은 디스크에 풀리지 않음 → 메모리에서만 존재

4

실시간 시간 관리

• Viewer는 실시간 타이머 작동

• 파일 사용 중에도 시간 차감

• 종료 시 남은 시간 저장

5

검증 실패 시

시간 초과, 권한 없음, 다른 회사 키 사용 시:

"파일을 열 수 없습니다" 메시지 출력

원본 정보는 단 1바이트도 노출되지 않음
